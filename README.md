## 🎨 Customization

### Thay đổi màu sắc:
```css
:root {
  --primary-dark: #124E66;
  --primary-green: #2E8B57;
  /* Cập nhật các biến CSS khác */
}
```

### Thay đổi fonts:
```css
:root {
  --font-primary: 'YourFont', sans-serif;
  --font-heading: 'YourHeadingFont', sans-serif;
}
```

### Thêm animations:
```html
<!-- Sử dụng AOS attributes -->
<div data-aos="fade-up" data-aos-delay="100">Content</div>
```
